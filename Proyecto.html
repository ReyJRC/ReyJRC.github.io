El Proyecto

Este proyecto propone el desarrollo de un sistema innovador que permite cargar un teléfono móvil utilizando la energía mecánica generada al pedalear una bicicleta. La iniciativa surge de la necesidad de aprovechar fuentes de energía limpias y accesibles, transformando el movimiento en una oportunidad para generar electricidad de manera ecológica y eficiente. Así, promovemos el uso de tecnologías sostenibles y la aplicación práctica de la física en soluciones reales para la vida cotidiana.

Materiales utilizados:

  • USB para carros
  • Condensador de 1000 microfaradios
  • Cable de teléfono
  • Motor reductor
  • Diodo 1N4004
  • Una llanta

Funcionamiento del sistema:

Al pedalear, se activa un generador (motor reductor) acoplado a la rueda de la bicicleta.

El generador convierte la energía mecánica en corriente eléctrica.

La corriente pasa por un diodo 1N4004, que evita la inversión de polaridad y protege el circuito.

Un condensador de 1000 microfaradios estabiliza la corriente para asegurar una carga constante.

Finalmente, el voltaje se regula y se conecta al USB para cargar el teléfono móvil de forma segura.

Gracias a este sistema, cada vez que pedaleas tu bicicleta, generas la energía suficiente para cargar tu dispositivo móvil. De esta manera, se fomenta el uso de fuentes alternativas de energía y se demuestra cómo la física puede integrarse en soluciones sostenibles y útiles para la vida diaria.
